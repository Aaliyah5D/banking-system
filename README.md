# banking-system
A Java console application simulating a basic banking system, demonstrating OOP principles, encapsulation, inheritance, and unit testing with JUnit 5.

 # Overview
This project is a Greenfield Java application that models a simple banking system. It was built to demonstrate core Object-Oriented Programming concepts including encapsulation, inheritance, and polymorphism, alongside documented design and comprehensive unit testing.

# Design
The system is built around a base BankAccount class, extended by SavingsAccount and ChequeAccount. All sensitive account data is encapsulated, with business logic enforced through controlled methods.

# Concepts Covered

Encapsulation — private fields with validated getters/setters
Inheritance — SavingsAccount and ChequeAccount extend BankAccount
Design Documentation — UML class diagram and design decisions included
Unit Testing — JUnit 5 tests covering core logic and edge cases

🛠️ Tech Stack

Java 17
JUnit 5
Maven
